# 🌐 Web Development Roadmap (Frontend + Backend)

> **Timeline:** 6–8 months | **Level:** Beginner to Intermediate

---

## Phase 1 — Foundations (Month 1–2)

### HTML
- Structure of a webpage, tags, attributes
- Forms, tables, semantic HTML
- ⏱️ Timeline: 1 week
- 📚 Resources:
  - [MDN HTML Docs](https://developer.mozilla.org/en-US/docs/Web/HTML)
  - [freeCodeCamp HTML Course](https://www.freecodecamp.org)

### CSS
- Selectors, box model, flexbox, grid
- Responsive design, media queries
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [CSS Tricks](https://css-tricks.com)
  - [Kevin Powell YouTube](https://www.youtube.com/@KevinPowell)

### JavaScript Basics
- Variables, functions, loops, DOM manipulation
- Events, fetch API, ES6+
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [javascript.info](https://javascript.info)
  - [Eloquent JavaScript (free book)](https://eloquentjavascript.net)

---

## Phase 2 — Frontend (Month 3–4)

### React.js
- Components, props, state, hooks
- React Router, Context API
- ⏱️ Timeline: 4 weeks
- 📚 Resources:
  - [Official React Docs](https://react.dev)
  - [Scrimba React Course](https://scrimba.com)

### Version Control
- Git basics, branching, merging, GitHub
- ⏱️ Timeline: 3 days
- 📚 Resources:
  - [Pro Git Book (free)](https://git-scm.com/book/en/v2)
  - [GitHub Skills](https://skills.github.com)

### Mini Project 🛠️
- Build a **personal portfolio website** using HTML, CSS & React

---

## Phase 3 — Backend (Month 5–6)

### Node.js + Express
- REST APIs, middleware, routing
- Authentication with JWT
- ⏱️ Timeline: 4 weeks
- 📚 Resources:
  - [Node.js Docs](https://nodejs.org/en/docs)
  - [The Odin Project](https://www.theodinproject.com)

### Databases
- SQL basics with PostgreSQL or MySQL
- NoSQL basics with MongoDB
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [SQLZoo](https://sqlzoo.net)
  - [MongoDB University](https://university.mongodb.com)

### Mini Project 🛠️
- Build a **full-stack blog app** with login, posts & comments

---

## Phase 4 — Polish & Deploy (Month 7–8)

### Topics
- Deployment with Vercel / Netlify / Render
- Environment variables, CORS, security basics
- Basic testing (Jest)
- ⏱️ Timeline: 3 weeks

### Final Project 🚀
- Build a **full-stack e-commerce or task manager app** and deploy it live

---

## ✅ Checklist

- [ ] HTML & CSS basics done
- [ ] JavaScript fundamentals done
- [ ] React project built
- [ ] Node + Express API built
- [ ] Database connected
- [ ] Full-stack app deployed
