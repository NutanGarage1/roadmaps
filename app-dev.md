# 📱 App Development Roadmap

> **Timeline:** 4–6 months | **Level:** Beginner to Intermediate

---

## Choose Your Path

| Path | Language | Best For |
|------|----------|----------|
| Android Native | Kotlin | Android-only apps |
| Cross-Platform | Flutter (Dart) | Android + iOS together |
| Cross-Platform | React Native | JS developers |

> 💡 **Recommended for beginners:** Flutter — one codebase, runs everywhere.

---

## Path A — Flutter (Cross-Platform)

### Phase 1 — Dart Basics (2 weeks)
- Variables, functions, OOP in Dart
- 📚 [Dart Language Tour](https://dart.dev/guides/language/language-tour)

### Phase 2 — Flutter Basics (Month 1–2)
- Widgets: Stateless vs Stateful
- Layouts: Column, Row, Stack, ListView
- Navigation & routing
- ⏱️ Timeline: 4 weeks
- 📚 Resources:
  - [Flutter Docs](https://flutter.dev/docs)
  - [Flutter & Dart — Udemy (Angela Yu)](https://www.udemy.com/course/flutter-bootcamp-with-dart/)

### Phase 3 — State Management (Month 3)
- setState, Provider, Riverpod or GetX
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Riverpod Docs](https://riverpod.dev)

### Phase 4 — Backend Integration (Month 4)
- REST API calls with `http` or `dio`
- Firebase: Auth, Firestore, Storage
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Firebase Flutter Docs](https://firebase.google.com/docs/flutter/setup)

---

## Path B — Android Native (Kotlin)

### Phase 1 — Kotlin Basics (2 weeks)
- Syntax, null safety, coroutines
- 📚 [Kotlin Docs](https://kotlinlang.org/docs/home.html)

### Phase 2 — Android Basics (Month 1–2)
- Activities, Fragments, Layouts (XML)
- RecyclerView, Intents, Permissions
- 📚 [Android Developers Docs](https://developer.android.com/docs)

### Phase 3 — Architecture & Jetpack (Month 3–4)
- MVVM architecture, ViewModel, LiveData
- Room database, Navigation Component
- 📚 [Android Jetpack](https://developer.android.com/jetpack)
