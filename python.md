# 🐍 Python Roadmap

> **Timeline:** 2–4 months | **Level:** Beginner to Advanced

---

## Phase 1 — Python Basics (Month 1)

### Core Syntax
- Variables, data types, operators
- Conditionals, loops, functions
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Python.org Docs](https://docs.python.org/3/)
  - [Automate the Boring Stuff (free)](https://automatetheboringstuff.com)

### Data Structures in Python
- Lists, tuples, sets, dictionaries
- List comprehensions, lambda functions
- ⏱️ Timeline: 1 week

### File Handling & Modules
- Reading/writing files, os & sys modules
- pip, virtual environments
- ⏱️ Timeline: 1 week

---

## Phase 2 — Intermediate Python (Month 2)

### Object-Oriented Programming
- Classes, objects, inheritance, polymorphism
- Dunder methods, decorators
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Real Python OOP](https://realpython.com/python3-object-oriented-programming/)

### Error Handling & Testing
- Try/except, custom exceptions
- unittest, pytest basics
- ⏱️ Timeline: 1 week

### Libraries to Know
- `requests` — HTTP calls
- `json` — data parsing
- `datetime` — date/time handling
- ⏱️ Timeline: 1 week

---

## Phase 3 — Advanced Python (Month 3–4)

### Generators & Iterators
- yield, itertools, lazy evaluation
- ⏱️ Timeline: 1 week

### Concurrency
- Threading, multiprocessing, asyncio basics
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Real Python Async](https://realpython.com/async-io-python/)

### Mini Projects 🛠️
- CLI to-do app
- Web scraper using `BeautifulSoup`
- REST API with `FastAPI` or `Flask`

---

## ✅ Checklist

- [ ] Python syntax & data structures done
- [ ] OOP concepts understood
- [ ] At least 2 mini projects built
- [ ] Comfortable with pip & virtual environments
