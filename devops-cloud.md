# ⚙️ DevOps & Cloud Roadmap

> **Timeline:** 4–6 months | **Level:** Intermediate

---

## Phase 1 — Linux & Networking Basics (Month 1)

### Linux
- File system, permissions, shell scripting
- Common commands: grep, awk, sed, cron
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Linux Journey](https://linuxjourney.com)
  - [The Linux Command Line (free book)](https://linuxcommand.org/tlcl.php)

### Networking Basics
- TCP/IP, DNS, HTTP/HTTPS, ports
- SSH, firewalls, load balancers
- ⏱️ Timeline: 1 week

---

## Phase 2 — Version Control & CI/CD (Month 2)

### Git Advanced
- Branching strategies, rebasing, cherry-pick
- GitHub Actions basics
- ⏱️ Timeline: 1 week

### CI/CD Pipelines
- GitHub Actions, Jenkins basics
- Build → Test → Deploy pipeline
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [GitHub Actions Docs](https://docs.github.com/en/actions)

---

## Phase 3 — Containers & Orchestration (Month 3–4)

### Docker
- Images, containers, Dockerfile, volumes
- Docker Compose for multi-container apps
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Docker Docs](https://docs.docker.com)
  - [TechWorld with Nana YouTube](https://www.youtube.com/@TechWorldwithNana)

### Kubernetes
- Pods, services, deployments, namespaces
- kubectl, Helm basics
- ⏱️ Timeline: 3 weeks

---

## Phase 4 — Cloud Platforms (Month 5–6)

### Pick One Cloud
- **AWS** — EC2, S3, RDS, Lambda, IAM
- **GCP** — Compute Engine, Cloud Storage, BigQuery
- **Azure** — VMs, Blob Storage, Azure DevOps
- ⏱️ Timeline: 4 weeks
- 📚 Resources:
  - [AWS Free Tier](https://aws.amazon.com/free/)
  - [A Cloud Guru](https://acloudguru.com)
  - [Google Cloud Skills Boost](https://cloudskillsboost.google)

### Infrastructure as Code
- Terraform basics
- ⏱️ Timeline: 2 weeks
