# 📊 Data Analyst Roadmap

> **Timeline:** 3–4 months | **Level:** Beginner to Intermediate

---

## Phase 1 — Foundations (Month 1)

### Excel / Google Sheets
- Formulas, pivot tables, VLOOKUP, charts
- Data cleaning basics
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Excel for Beginners — Leila Gharani YouTube](https://www.youtube.com/@LeilaGharani)

### Statistics Basics
- Mean, median, mode, standard deviation
- Probability, distributions, hypothesis testing
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Khan Academy Statistics](https://www.khanacademy.org/math/statistics-probability)

---

## Phase 2 — SQL & Databases (Month 2)

### SQL
- SELECT, WHERE, GROUP BY, JOINs, subqueries
- Window functions, CTEs
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Mode SQL Tutorial](https://mode.com/sql-tutorial/)
  - [SQLZoo](https://sqlzoo.net)
  - [LeetCode SQL Problems](https://leetcode.com/problemset/database/)

---

## Phase 3 — Data Visualization (Month 3)

### Power BI / Tableau
- Connecting data sources, building dashboards
- Filters, slicers, calculated fields
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Power BI Docs](https://learn.microsoft.com/en-us/power-bi/)
  - [Tableau Public (free)](https://public.tableau.com)

### Python for Analysis (Optional but recommended)
- Pandas, Matplotlib, Seaborn
- ⏱️ Timeline: 2 weeks
- 📚 Resources:
  - [Kaggle Python Course (free)](https://www.kaggle.com/learn/python)

---

## Phase 4 — Projects & Portfolio (Month 4)

### Mini Projects 🛠️
- Sales dashboard in Power BI
- SQL analysis on a public dataset (e.g. COVID, IPL, Netflix data)
- EDA (Exploratory Data Analysis) in Python

### Where to Find Datasets
- [Kaggle Datasets](https://www.kaggle.com/datasets)
- [Google Dataset Search](https://datasetsearch.research.google.com)
- [data.gov.in](https://data.gov.in) — India specific

---

## ✅ Checklist

- [ ] Excel pivot tables & formulas done
- [ ] SQL JOINs & window functions mastered
- [ ] Dashboard built in Power BI or Tableau
- [ ] At least 2 projects in portfolio
