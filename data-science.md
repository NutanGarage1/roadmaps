# 🔬 Data Science Roadmap

> **Timeline:** 5–7 months | **Level:** Intermediate

> 💡 Complete the Python & Data Analyst roadmaps first for best results.

---

## Phase 1 — Math & Stats for DS (Month 1)

### Topics
- Linear algebra: vectors, matrices, dot products
- Calculus: derivatives, gradients (for ML)
- Probability & Bayes theorem
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [3Blue1Brown Linear Algebra](https://www.youtube.com/@3blue1brown)
  - [StatQuest YouTube](https://www.youtube.com/@statquest)

---

## Phase 2 — Python for Data Science (Month 2)

### Libraries
- **NumPy** — arrays & numerical computing
- **Pandas** — data manipulation & analysis
- **Matplotlib & Seaborn** — data visualization
- ⏱️ Timeline: 3 weeks
- 📚 Resources:
  - [Kaggle Pandas Course (free)](https://www.kaggle.com/learn/pandas)
  - [Kaggle Data Visualization (free)](https://www.kaggle.com/learn/data-visualization)

---

## Phase 3 — Machine Learning (Month 3–4)

### Supervised Learning
- Linear & logistic regression
- Decision trees, Random Forest, XGBoost
- SVM, KNN
- ⏱️ Timeline: 4 weeks

### Unsupervised Learning
- K-Means clustering, PCA
- ⏱️ Timeline: 2 weeks

### Tools
- **Scikit-learn** — model building & evaluation
- **Jupyter Notebook** — experimentation
- 📚 Resources:
  - [Hands-On ML with Scikit-Learn (book)](https://www.oreilly.com/library/view/hands-on-machine-learning/9781492032632/)
  - [fast.ai (free)](https://www.fast.ai)

---

## Phase 4 — Deep Learning Basics (Month 5–6)

### Topics
- Neural networks, activation functions, backprop
- CNNs for image data, RNNs for sequences
- ⏱️ Timeline: 4 weeks
- 📚 Resources:
  - [Deep Learning Specialization — Coursera (Andrew Ng)](https://www.coursera.org/specializations/deep-learning)
  - [PyTorch Docs](https://pytorch.org/tutorials/)

---

## Phase 5 — Projects & Deployment (Month 7)

### Mini Projects 🛠️
- House price prediction (regression)
- Sentiment analysis on Twitter/Amazon data
- Image classifier using CNN
- Deploy a model with Flask or Streamlit

### Platforms
- [Kaggle Competitions](https://www.kaggle.com/competitions)
